# Microsoft Fabric Toolbox

This directory contains utility scripts, automation tools, and helper functions for working with Microsoft Fabric.

## What's in the Toolbox?

The toolbox contains practical utilities that help with:

- **Deployment**: Scripts for deploying Fabric items
- **Configuration**: Tools for managing configurations
- **Monitoring**: Scripts for monitoring workspaces and items
- **Migration**: Utilities for migrating between environments
- **Backup & Restore**: Tools for backing up Fabric artifacts
- **Automation**: Scripts for automating repetitive tasks

## Categories

Tools are organized into categories:

### Deployment Tools
Scripts and utilities for deploying Fabric items across environments

### Configuration Management
Tools for managing workspace configurations, connection strings, and parameters

### Monitoring & Logging
Utilities for monitoring workspace usage, performance, and logs

### Data Management
Tools for data lifecycle management, archival, and cleanup

### Development Utilities
Helper scripts for local development and testing

## How to Use Tools

1. Review the tool documentation in its specific directory
2. Check prerequisites and dependencies
3. Configure any required parameters or settings
4. Run the tool according to its usage instructions
5. Review output and logs

## Prerequisites

Many tools require:

- Python 3.8 or higher
- Azure CLI (for some deployment tools)
- Microsoft Fabric workspace access
- Appropriate permissions
