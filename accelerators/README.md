# Microsoft Fabric Accelerators

This directory contains complete project templates and accelerators to help you quickly start building Microsoft Fabric solutions.

## What are Accelerators?

Accelerators are pre-built templates that provide a complete starting point for common Microsoft Fabric scenarios. Each accelerator includes:

- Complete project structure
- Sample code and configurations
- Setup instructions
- Best practices implementation
- Documentation

## Available Accelerators

Browse the subdirectories to find accelerators for various scenarios:

- Data pipelines
- Lakehouse implementations
- Data warehousing
- Real-time analytics
- And more...

## How to Use an Accelerator

1. Navigate to the accelerator directory that matches your needs
2. Read the README.md in that directory for specific instructions
3. Copy the template to your workspace
4. Follow the setup instructions
5. Customize for your specific requirements

## Contributing

Have an accelerator to share? Please see the main [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines on contributing accelerators.

## Structure

Each accelerator should follow this structure:

```
accelerator-name/
├── README.md           # Detailed documentation
├── src/               # Source code
├── config/            # Configuration files
└── examples/          # Usage examples
```
